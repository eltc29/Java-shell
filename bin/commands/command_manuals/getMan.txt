 get URL
 
 Retrieve the file at that URLand add it to the current working directory.
 Example1:get http://www.cs.cmu.edu/ spok/grimmtmp/073.txt Will get the contents of the file,
 i.e. 073.txt and create a file called 073.txt with the contents in the currentworking directory.
 Example2:get http://www.ub.edu/gilcub/SIMPLE/simple.htmlWill get the contents of the file,
 i.e. simple.html (raw HTML) and create a file called simple.html with thecontents in the
 current working directory.
